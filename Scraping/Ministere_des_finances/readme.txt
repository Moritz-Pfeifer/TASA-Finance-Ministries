1.Scraping economie.gouv.fr
For scraping economie.gouv.fr run "python egf_scraper.py". All the date will be in economie.gouv.fr file.

2.Scraping records to JSON file
There is a column in the database "extract_to_json"(True/False(1/0)). Please mark the entries which you want to extract with "1" and run "python egf_scraper_to_json.py".

All the data from archive.org is in wayback_economie.gouv.fr
Dosiers PDF files are in WB_PDF folder
